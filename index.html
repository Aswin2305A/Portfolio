A fully client-side tool that crawls Wikipedia's live link graph and computes PageRank from first principles, rendering the result as an interactive force-directed network — no backend, no libraries.

  • Implements PageRank from scratch via power iteration (40 rounds, 0.85 damping factor) directly in the browser, with no server or external graph libraries.

  • Renders a custom force-directed graph on HTML5 Canvas — 120 iterations of repulsion/spring physics — with pan, zoom, and a gold-highlighted PageRank leaderboard.
